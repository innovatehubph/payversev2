# Get Circulating Coins

Retrieve information about circulating coins and their balances

### Quick Example

Copy

```
curl -X 'POST' \
  'https://api.pay-gram.org/{token}/GetCirculatingCoins' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json'
```

### Parameters

token

required

`string (uuid)`

Authentication token for API access

### Response

200

Success

```
{
  "success": true,
  "type": 16,
  "data": {
    // Response data varies by endpoint
  }
}
```
